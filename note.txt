//download.py
codespace 에서 작업하고 있고, github pages 에서 웹페이지를 보여주려 한다.
tickers.json 파일에는 yahoo finance 의 티커 정보가 개행문자를 구분자로 하여 저장되어 있다. 
해당 티커의 1990-현재까지 일별 데이터를 다운로드 하자.
index.html 파일에 다운로드한 티커별 차트를 위에서 부터 아래로 순서대로 차트별로 600X400 px 크기로 보여주자.
차트 제목은 해당 티커의 yahoo finance 상 표현되는 이름이다.
차트를 그릴때 그리는 부분의 일일데이터 값이 conditions.json 파일의 조건을 만족하면 해당 차트 부분은 빨간색으로 그린다. 만약 만족하지 않으면 검은색으로 그린다.
conditions.json 파일의 조건은 자연어로 씌어있으니 코드에서 해당 자연어를 읽어 조건으로 처리할수 있어야 한다.
index.html 의 각 티커별 차트 아래에는 버튼이 5개(3m, 6m, 1y, 5y, all) 있다.
3m 을 클릭하면 해당 티커의 3개월간 일별 그래프를 보여준다.
6m 을 클릭하면 해당 티커의 6개월간 일별 그래프를 보여준다.
1y 을 클릭하면 해당 티커의 1년간 일별 그래프를 보여준다.
5y 을 클릭하면 해당 티커의 5년간 일별 그래프를 보여준다.
all 을 클릭하면 해당 티커의 전체 기간의 일별 그래프를 보여준다.

//사용법
1. download.py 파이선실행
2. index.html 파일 git add/commit/push
3. 웹브라우저에서 https://cath7512.github.io/assets/index.html 실행


git add index.html
git commit -m "index.html 최신 차트 반영"
git push
